# Mycelium Frontend

This repository contains code for the Mycelium SDK.

## Runtime Endpoint

The SDK logs to `https://www.myceliums.dev/api` by default.

For local development, set `MYCELIUM_API_ENDPOINT` before starting an
instrumented service:

```bash
MYCELIUM_API_ENDPOINT=http://localhost:8000/api npm run start
```
